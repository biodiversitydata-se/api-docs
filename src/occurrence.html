    Endemic Species list - https://records.biodiversitydata.se/ws/explore/endemic/species   https://records.biodiversitydata.se/ws/explore/endemic/species
    GET JSON
    • GET
    • JSON

    List endemic species within an area (* indicates required params)

    q * String

    The initial query. q=*:* will query anything, q=macropus will do a free text search for macropus, q=kingdom:Fungi will search for records with a kingdom of Fungi. For a listing of the fields that can be queried in a q=INDEXEDFIELD:VALUE fashion, see /index/fields. This will restrict the records that are considered in then region. For example this could allow people to restrict the service to endemic birds.
    fq String

    Filters to be applied to the original query. These are additional params of the form fq=INDEXEDFIELD:VALUE e.g. fq=kingdom:Fungi. See https://records.biodiversitydata.se/ws/index/fields for all the fields that a queryable.
    wkt * String

    The Well Known Text Area in which to provide the endemic species.
    facets * String

    The field on which to based the species list. This can be taxon_concept_lsid or species_guid

    Examples

    • Edit
      List endemics (JSON)

      List (in JSON) all the endemic species recorded within a small area in London. (Note, wkt POLYGON=lon,lat and not lat,lon).

      URL: https://records.biodiversitydata.se/ws/explore/endemic/species?wkt=POLYGON((0.12+51.50,0.13+51.50,0.13+51.51,0.12+51.51,0.12+51.50))&q=*:*&facets=species_guid

    Collectory breakdown - https://records.biodiversitydata.se/ws/breakdown/{collectorytype}/{uid}   https://records.biodiversitydata.se/ws/breakdown/{collectorytype}/{uid}
    GET JSON
    • GET
    • JSON

    Performs a breakdown of the supplied collectory type.

    max Integer

    The maximum number of names to include in the breakdown. When this value is supplied the breakdown will traverse up the taxonomic hierarchy until a maximum of max names is discovered. Thus the rank of the breakdown is calculated by this service and should not be provided.
    name String

    The scientific name to limit the breakdown to - needs a valid rank to be supplied
    rank String

    The rank at which to perform the breakdown. If a name is specified in conjunction to this the breakdown will be performed at the next level further limiting the query.

    Examples

    • Edit
      Breakdown collectory and return counts

      Breakdown collectory by phylum

      URL: https://records.biodiversitydata.se/ws/breakdown/dataResources/dr495?rank=phylum

    Occurrence download - https://records.biodiversitydata.se/ws/occurrences/index/download   https://records.biodiversitydata.se/ws/occurrences/index/download
    GET CSV SHP
    • GET
    • CSV
    • SHP

    Occurrence download - limited to first 500,00 records.
    Downloads of the following formats are supported:

    • CSV
    • TSV
    • SHAPE

    The output format is determined based on the fileType param.

    The download will return a ZIP file that contains the data in the requested format and a Readme and citataion.csv.
    Please be aware that a shape file download will be large. Each of the fields is included as a feature. It is advised to select a subset of information to be included.

    q * String

    Query of the form field:value e.g. q=genus:Macropus or a free text search e.g. q=Macropus
    fq String

    Filters to be applied to the original query. These are additional params of the form fq=INDEXEDFIELD:VALUE e.g. fq=kingdom:Fungi. See https://records.biodiversitydata.se/ws/index/fields for all the fields that a queryable.
    lat Double

    The decimal latitude to limit records to. Use with lon and radius to specify a "search" circle
    lon Double

    The decimal latitude to limit records to. Use with lon and radius to specify a "search" circle
    radius Double

    The radius in which to limit records (relative to the lat, lon point). Use with lat and lon to specify a "search" circle.
    wkt String

    The polygon area in which to limit records. For information on Well known text
    file String

    The name of the file to produce. This will default to data.

    email String

    The email address of the user performing the download.
    reason String

    A user supplied description of the reason for the download
    fields String

    A CSV list of fields to include in the download. This can be a stored index field name OR a Darwin Core Field name that is also a stored index field. There is a default list (documented in a Google Doc) that is included if no value is supplied. A list of index fields can be found here: https://records.biodiversitydata.se/ws/index/fields.
    extra String

    a CSV list of fields in include in addition to the "fields". This is useful if you would like to make use of the default fields and include extra fields.
    reasonTypeId * Integer

    A reason code for the download. See reasons for valid values.
    fileType String

    The file format for the download. One of csv, tsv or shp. When no value is supplied csv is assumed.

    sourceTypeId Integer

    A source type code for the download. This indicates which system requested the download. See sources for valid values.
    qa String

    A CSV list of record issues to include in the download. See assertions for possible values to include. When no value is supplied all the record issues are included. To include NO record issues set this value to "none". To include ALL INCLUDING EMPTY record issue columns set this value to "includeall".
    esc String

    The Escape character to use for a CSV file. This will default to " in order to produce an Excel compliant CSV file.
    sep String

    The separator character to use for a CSV file. This will default to , .
    dwcHeaders Boolean

    Use DwC field names in the download where available. The default value is "false".

    Examples

    • Edit
      Disable Record Issues in download

      A "test" (recordings of genus Vulpes recording from a 10k radius in London) in a download that does not include record assertions.

      URL: https://records.biodiversitydata.se/ws/occurrences/index/download?reasonTypeId=10&q=*:*&fq=genus:Vulpes&lat=51.5074&lon=0.1278&radius=10.0&qa=none

    facet String

    pageSize Integer

    Number of records to return

    lon Double

    The polygon area in which to limit records. For information on Well known text
    ENV String

    Additional parameters. e.g. ENV=color%3Acd3844%3Bsize%3A3%3Bopacity%3A0.8.

    • color - hex RGB values. e.g. colour:cd3844.
    • size - radius of points in pixels.
    • opacity - opacity value 0 - 1.
    • sel - fq parameter applied to CQL_FILTER. Matching occurrences will be highlighted on the map in a Red circle.
    • uncertainty - presence of the uncertainty parameter draws uncertainty circles to a fixed maximum of 30km.
    • colormode - facet colouring type.
      • -1 (default) use color value.
      • grid map as density grid. Grid cells drawn are not restricted to within any query WKT parameters.
      • facetname colour as categories in a facet.
      • facetname,cutpoints colour as range in a facet using the supplied comma separated cutpoints. 4 to 10 values are required. Include minimum and maximum. Minimum and maximum values do not need to be accurate. e.g. colormode:year,1800,1900,1950,1970,1990,2010


    Please see the examples for more available ENV parameters
